在循环中可以使用break;语句终止整个循环
     的执行
如果循环是采用break;语句结束的，则循环
    结束后循环变量的数值就是执行break;
		语句时的数值
在循环中可以使用continue;语句跳转到
    大括号的末尾
continue;语句只会影响当次大括号里面
    语句的执行

假设有如下for循环语句
for (1;2;3) {
		4
}
它按照如下方式分组执行
[1,2][4,3,2][4,3,2]....[4,3,2]
除了第一组外后面的每组都一样
正常情况下for循环会在两组之间结束
如果某组中最后编号为2的逻辑表达式
    结果是假则循环结束，否则循环
		继续
如果一个循环是正常结束的则标号为
    2的逻辑表达式结果一定是假
for循环中大括号里面的语句有可能不
    执行

复杂的循环问题可以先分组，为每组找到
    一个代表数字。
采用一个循环描述所有组代表数字的变化
    过程
再编写一个循环负责处理某一组中的所有
    工作
把后一个循环写在前一个循环内部，使用
    这两个循环一起解决问题
这种使用循环的方式叫循环嵌套
循环嵌套中每个循环使用自己的循环变量
内循环的开始数字和结束数字可能需要
    根据外循环的循环变量计算得到

for循环的多种写法
1.省略大括号
2.小括号中可以使用，连接多个语句
3.在小括号中临时声明循环变量(C99规范
		才支持)。循环结束后循环变量不可
    再用
4.小括号里只保留逻辑表达式
5.在小括号后面直接用;结束（空语句）
6.省略小括号中间的逻辑表达式（死循环）

while关键字也可以用来编写循环
while循环和for循环是完全可以互相替代的
优先选择采用for关键字编写循环
while循环的格式如下
while (布尔值) {
		反复执行的语句
}
可以认为每当布尔值为真时大括号内部
     的语句执行一次

假设有如下while循环语句
while (1) {
		2
}
它按照如下方式分组执行
[1][2,1][2,1][2,1]....[2,1]
除了第一组以外其他组都一样
每组最后一步都是编号为1的逻辑表达式，
    这个逻辑表达式的结果决定循环
		是否要继续。如果结果是真则继续
		循环，如果结果是假则结束循环。
大括号里面的语句有可能不执行

while语句也可以实现空语句和死循环
while语句实现死循环的方式如下
while (1) {
		....
}

while循环中也可以使用break;和continue;
    语句

do...while语句也可以用来实现循环，这种
    循环中大括号里面的语句至少会执行
		一次
do...while循环中一定要在小括号后面加；
假设有如下语句
do {
		2
} while (1);
它按照如下方式分组执行
[2,1][2,1][2,1]....[2,1]
这种分组方式相当于把while循环中第一组
    取消

scanf函数调用语句可以当作整数使用，如果
    是0则表示没有读到有效数据，如果是
		其他数字表示得到有效数据了

goto也是一种流程控制语句，它可以指定任何
    语句成为下一条语句
不要使用goto语句

可以通过数组使用内存中一组连续的存储区
这组存储区的类型必须一样
数组有三个要素，一个是数组名称，另一个
    是数组中存储区的类型，存储区的
		个数
数组也必须先声明然后才能使用。
数组不能被当成一个整体使用，每次只能
    使用其中的一个存储区
数组中每个存储区有一个编号，这个编号
    叫下标。下标的范围从0到个数减一
		为止。
地址数据加减整数时不是按照数学规则
    计算的，整数1代表了一个存储区
		的大小

如果希望按顺序依次使用数组中的每一个
    存储区需要使用for循环，循环中
		循环变量依次代表每一个存储区的
		下标

数组初始化
1.数组需要使用多个初始化数据，把所有
    初始化数据写在一对大括号中，不同
		数据之间使用，分开
2.前边的初始化数据用来初始化下标小的
    存储区
3.多余的初始化数据自动被丢弃
4.没有对应初始化数据的存储区自动被
    初始化成0
5.如果每个存储区都有对应的初始化数据
    则可以省略数组声明中的存储区个数

数组名称代表数组中第一个存储区的地址，
    它不可以被修改（不能被赋值）

对数组名称进行sizeof计算结果是数组中
    包含的字节个数

数组一旦存在其中所包含的存储区个数
    不可改变
C99规范中支持变长数组，这种数组可以在
    程序运行时数组声明前确定数组中
		存储区的个数
变长数组不可以初始化
