变量可以用来代表一组相关数字（任何时候
    只使用其中的一个数字，数字之间
    有关联）
    
内存中的每个存储区有一个编号（地址），
    不同存储区的地址不同
只有先找到存储区的地址然后才能找到
    存储区

程序中可以在变量名称前加符号&，这样
    可以直接找到变量对应存储区的地址
    
scanf是一个C语言提供的标准函数，它可以
    把用户在键盘上输入的数字直接记录
    在存储区中
为了使用这个标准函数需要把stdio.h文件
    的内容包含在源文件中
scanf函数调用语句中必须使用存储区地址
    表示存储区
scanf函数调用语句中应该使用占位符表示
    存储区的类型
不要在scanf函数调用语句中加\n字符
双引号中不是占位符的内容需要在键盘
     上按顺序原样输入

在读数据的时候如果前面输入的内容没有
    处理则后面输入的内容就没有机会被
    处理
使用如下两条语句可以把用户某次输入的
    错误数据处理掉
scanf("%*[^\n]");
scanf("%*c");
    
C语言中把数字分成几组，每组中的数字
    之间只有数值的区别。每组数字叫做
    一个数据类型。

不同数据类型可能包含同样的数值

字符类型是一种数据类型
字符类型用char表示
字符类型中包含256个不同的字符
计算机中使用整数表示字符，每个字符
    有一个对应的整数
字符数据在C语言中采用如下方式表示
    （'a','&','>'等）    
字符数据有两种表示方式（身份），分别
    是字符身份和整数身份
ASCII码表中列出256个字符两种表示方式
    之间的对应关系
'a'    --        97
'A'    --        65
'0'    --        48
所有字符分成两组，每组128个
其中一组的整数表示方式在所有计算机上
    都一样，范围从0到127
另外一组的整数表示方式在不同计算机上
    可能不同。范围可能从-128到-1或者
    从128到255 
    
无符号字符类型也包含256个不同的字符
无符号字符类型用unsigned char表示
这种类型中字符的整数表示方式固定从
    0到255
    
ASCII码表中所有小写英文字母的整数表示
    方式之间是连续的，'a'的整数表示
    方式最小，'z'的整数表示方式最大
以上结论同样适用于所有大写英文字母和
    阿拉伯数字字符
    
't' - 'a'   等于   'T' - 'A'
'd' - 'a'   等于   '3' - '0'

'\n'    换行字符
'\r'    回车字符
'\t'    制表符
'\\'    代表\字符
'\''    代表'
'\"'    代表"
    
短整数类型用short表示
这个类型一共包含65536个不同的整数，
    范围从-32768到32767
无符号短整数类型用unsigned short表示
无符号短整数类型同样包含65536个整数，
    范围从0到65535

长整数类型用long或long int表示
这个数据类型中包含2的32次方个不同的
    整数。范围从-2的31次方到2的31次方
    减一
在不同的计算机上所包含的整数个数不固定
无符号长整数用unsigned long或unsigned
    long int表示
范围从0到2的32次方减一

整数类型用int表示，在我们的计算机上
    和长整数一样
无符号整数类型用unsigned int表示，在
    我们的计算机上和无符号长整数一样

以上数据类型所表示的整数范围是重叠的，
    并且逐渐扩大

c语言中使用浮点类型表示带小数点的数值
浮点类型分为单精度浮点类型和双精度
    浮点类型
单精度浮点类型用float表示
双精度浮点类型用double表示
单精度浮点类型的精度低

程序中带小数点的数字默认类型是
    双精度浮点类型

在程序中不带小数点数字后面加u则表示
    这个数字是无符号整数类型，例如
    5u
在程序中带小数点数字后面加f表示
    这个数字是单精度浮点类型，例如
    4.67f

数据类型和占位符的对应关系
char                --   %c和%hhd
unsigned char       --   %c和%hhu
    %c对应字符表示方式，%hhd和%hhu
    对应整数表示方式
short               --   %hd
unsigned short      --   %hu
long                --   %ld
unsigned long       --   %lu
int                 --   %d
unsigned int        --   %u
float               --   %f和%g
double              --   %lf和%lg
    %f和%lf会保留小数点后无效的0

%nd也可以作为占位符使用，其中n代表
    一个整数。打印结果会栈n个位置。
    需要时会在真实内容前补充空格字符
%-nd也可以作为占位符，n同样代表一个
    整数。效果和%nd类似，区别是空格
    补充在右边。
%0nd也可以作为占位符，n同样代表一个
    整数。效果和%nd类似，区别是用
    字符'0'替换空格字符
%n.mf可以用来做占位符打印浮点数，
    打印结果一共占n个位置，其中小数点
    后面占m个位置
    
不同类型的存储区大小不同
内存分成大量同样大小的格子，每个格子
    叫一个字节。任何一个存储区都是由
    一个或多个相邻的字节拼凑成的。
存储区的大小指的是它里面包含的字节个数

sizeof关键字可以用来计算一个数据类型
    或存储区所包含的字节个数

char(unsigned char)     1个字节
short(unsigned short)   2个字节
long(unsigned long)     4个字节
int(unsigned int)       4个字节
float                   4个字节
double                  8个字节

在sizeof关键字的小括号里可以写一个
    计算公式，它可以把结果的大小
    计算出来
sizeof语句中对存储区内容的修改不会
    真正生效
